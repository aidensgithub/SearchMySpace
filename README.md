# SearchMySpace
A Twitter chatbot for searching Spaces.

To search for spaces enter /search and input your search keywords.

# Example:
`/search nft music`

# Additional features:
1) Filter by status of Space
`/search nft music *live`
`/search nft music *scheduled`
1) Filter by language
`/search nft music #fr`
`/search nft music #es`
